# Guardrail 04 - Cobertura de Fontes

Fontes devem ser reais, rastreáveis e úteis.

## Checks

- cada fonte tem URL clicável;
- fontes primárias foram priorizadas;
- fontes secundárias têm motivo claro;
- cada fonte registra limite quando relevante;
- cada node tem referências específicas;
- referências no JSON apontam para fontes declaradas;
- nenhum fato instável foi inventado;
- datas e versões foram verificadas quando o tema exigir.

Se a pesquisa falhar e for essencial, bloqueie a geração em vez de inventar.
